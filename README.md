# Backyard Bullies Wrestling Club — Website

Single-page website for **Backyard Bullies Wrestling Club** (Augusta, GA / CSRA).
Plain HTML + [Tailwind CSS](https://tailwindcss.com) (compiled to a static stylesheet) + a
little vanilla JS. Hosted on **GitHub Pages**, built and deployed automatically by GitHub Actions.

🔗 **Live:** https://www.backyardbullies-wc.com/

## Files

```
index.html                    The entire site (one page, scroll sections)
src/input.css                 Tailwind entry + custom CSS (source for the build)
tailwind.config.js            Tailwind theme (brand colors, fonts)
package.json                  Build scripts + Tailwind dev dependency
assets/tailwind.css           COMPILED output — generated by the build (git-ignored)
images/                       Logo, hero/cover, About photo, favicons, OG image
images/store/                 Team-store product shots used in the "Shop" montage
.github/workflows/deploy.yml  CI: build Tailwind + deploy to GitHub Pages
.nojekyll                     Tells Pages to serve files as-is
README.md                     This file
```

## Local development

You need [Node.js](https://nodejs.org) (v20+). Then:

```bash
npm install        # one-time: installs Tailwind
npm run build:css  # compile assets/tailwind.css once
# or, while editing styles/markup:
npm run watch:css  # rebuild on every change
```

Then open `index.html` (or serve the folder, e.g. `python3 -m http.server`).

> `assets/tailwind.css` is **git-ignored** — it's a build artifact. The site won't be styled
> until you run a build locally; in production GitHub Actions builds it fresh on every deploy.

## 💳 Membership & payments

There is no online checkout. Billing is handled by **emailed invoice** or **payment in
person**, and the buttons in the **Membership & Payments** section are simple call/text
links (`tel:+17062500186`).

- Prices are written directly in the markup — monthly tiers ($90 / $115 / $130 / $150 for
  2–5 practices per week) and the $15 open-mat / drop-in. Edit those values in `index.html`
  if rates change.

  the club admin for details.

## "From the Mat" — live Facebook feed

The **From the Mat** section embeds your Facebook Page's live feed using the official
[Facebook Page Plugin](https://developers.facebook.com/docs/plugins/page-plugin/) — no API
key, no token, no maintenance. New posts appear automatically.

- It renders only on a **real web domain** (e.g. the GitHub Pages URL), *not* over `localhost`
  or `file://`, and the visitor must allow Facebook's script. A harmless `404` from
  `/platform/plugin/page/logging/` may appear in the console on unregistered domains — ignore it.
- To point it at a different page, edit the `data-href` on the `fb-page` div in `index.html`.

## Deploy (GitHub Pages via Actions)

Already configured — **just push to `master`** and the site rebuilds and redeploys:

1. `.github/workflows/deploy.yml` runs on every push to `master`.
2. It installs Tailwind, compiles `src/input.css` → `_site/assets/tailwind.css`, assembles the
   site into `_site/`, and deploys that to GitHub Pages.
3. Pages **Settings → Build and deployment → Source** is set to **GitHub Actions**.

Watch runs under the repo's **Actions** tab. The live URL is printed on the deploy step.

### Custom domain (optional)

1. Add a `CNAME` file containing only your domain (e.g. `www.backyardbullies-wc.com`), and include it
   in the deploy (copy it into `_site/` in the workflow).
2. At your registrar, point DNS to GitHub Pages
   ([guide](https://docs.github.com/pages/configuring-a-custom-domain-for-your-github-pages-site)).
3. In **Settings → Pages**, set the custom domain and enable **Enforce HTTPS**.

> Also update the `<link rel="canonical">`, `og:url`/`og:image`, and JSON-LD URLs in
> `index.html` to match the final domain.

## Notes

- **Accessibility:** semantic landmarks, skip link, keyboard-operable mobile menu, alt text,
  labeled icon links, visible focus rings, and `prefers-reduced-motion` support are built in.
- **Styling:** Tailwind is compiled (no runtime CDN), so the production page ships one small,
  cached stylesheet.
